1.网络性能

`iperf` 是一款**开源、跨平台**的网络性能测试工具，核心用来：

- 测试 TCP/UDP 带宽（如千兆网卡、万兆交换机、公网 / 内网的真实传输速度）；
- 分析网络延迟、丢包率、抖动（UDP 场景）；
- 模拟单 / 多线程网络流量，定位网络瓶颈（如网卡、路由、防火墙）；
- 支持 IPv4/IPv6，是运维 / 网络工程师必备工具。

> 注意：常用版本分 `iperf2` 和 `iperf3`（推荐 iperf3，功能更全、兼容性更好）。

命令
服务器 # iperf3 -s

客户端测tcp # iperf3 -c 172.31.32.173

客户端测udp 指定带宽1000M# iperf3 -u  -b 1000M -c 172.31.32.173
