







在 Docker Desktop 中，若想快速打开同一个 Docker 容器的多个终端，可使用`docker exec`命令。具体操作步骤如下6：

1. 首先，通过`docker ps`命令查看正在运行的容器列表，找到目标容器的容器 ID 或容器名称。例如，假设目标容器的 ID 为`40c330755e61`。
2. 然后，在终端中执行`docker exec -it <容器ID或容器名称> /bin/bash`命令来打开一个新终端进入容器。如`docker exec -it 40c330755e61 /bin/bash`，即可进入该容器的一个新终端。
3. 若需要打开更多终端，重复步骤 2 即可。每次执行`docker exec -it`命令都会打开一个新的终端连接到同一个容器，且这些终端之间相互独立，不会出现输入同步的情况。
