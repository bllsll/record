C++ 智能指针是 **RAII（资源获取即初始化）** 思想的核心实现，用于自动管理动态内存（堆内存），避免内存泄漏、野指针、重复释放等问题。其本质是**封装了原始指针的类模板**，通过构造函数获取资源，析构函数（离开作用域时自动调用）释放资源，无需手动调用 `delete`。



C++11 及后续标准提供了 3 种核心智能指针（定义在 `<memory>` 头文件中）：`unique_ptr`、`shared_ptr`、`weak_ptr`，各自适用于不同场景。



## unique_ptr：独占式智能指针（最常用）

`unique_ptr` 是 **独占所有权** 的智能指针：同一时间，一个堆内存资源只能被一个 `unique_ptr` 持有，禁止拷贝（避免 double free），支持移动（所有权转移）。

默认情况下，`unique_ptr` 用 `delete`（或 `delete[]`）释放资源。若资源不是堆内存（如文件句柄、Socket），可自定义删除器。

### 1. 核心特性

- 独占性：不允许拷贝构造、拷贝赋值（函数被 `delete` 禁用）。
- 轻量级：无额外开销（大小与原始指针一致），效率和原始指针接近。
- 支持数组：需显式指定 `std::unique_ptr<T[]>`（析构时自动调用 `delete[]`）。
- 可定制删除器：支持自定义资源释放逻辑（如释放文件句柄、网络连接）

#### 创建 `unique_ptr`

禁止直接用原始指针赋值（如 `unique_ptr<int> p = new int(10);` 编译报错

```cpp
    // 1. 推荐：用 make_unique 创建（C++14+）
    unique_ptr<int> p1 = make_unique<int>(10); // 指向 int(10)
    unique_ptr<int[]> p2 = make_unique<int[]>(5); // 指向大小为5的int数组（值初始化为0）
    
    // 2. 不推荐：直接用构造函数（需手动传入 new 指针）
    unique_ptr<int> p3(new int(20));
```

```cpp
unique_ptr<int> p1 = make_unique<int>(10);
unique_ptr<int> p2 = move(p1); // 所有权转移给 p2，p1 变为空

cout << (p1 ? "p1 非空" : "p1 为空") << endl; // 输出 p1 为空
cout << *p2 << endl; // 输出 10

// 错误：禁止拷贝（编译报错）
// unique_ptr<int> p3 = p2; // 编译失败：拷贝构造被 delete
// p1 = p2; // 编译失败：拷贝赋值被 delete
```

### 适用场景

- 管理单个堆对象（如局部动态变量、函数返回动态对象）。

- 作为容器元素（如 `vector<unique_ptr<int>>`），避免拷贝开销。

- 独占资源的场景（如工厂模式返回的对象）。

  

## shared_ptr：共享式智能指针

`shared_ptr` 是 **共享所有权** 的智能指针：多个 `shared_ptr` 可同时持有同一个堆内存资源，内部通过 **引用计数** 管理生命周期。

### 1. 核心原理：引用计数

- 每个被 `shared_ptr` 管理的资源，都对应一个 **引用计数**（`use_count`），记录当前持有该资源的 `shared_ptr` 数量。
- 当创建一个新的 `shared_ptr` 指向该资源时，引用计数 +1。
- 当一个 `shared_ptr` 析构（离开作用域）或重置（`reset()`）时，引用计数 -1。
- 当引用计数变为 0 时，自动调用 `delete` 释放资源。

```cpp
// 1. 推荐：用 make_shared 创建
    shared_ptr<int> p1 = make_shared<int>(10);
    cout << "p1 引用计数：" << p1.use_count() << endl; // 输出 1

    // 2. 拷贝：引用计数 +1
    shared_ptr<int> p2 = p1; // 拷贝构造，use_count = 2
    shared_ptr<int> p3(p1);  // 拷贝构造，use_count = 3
    cout << "p1 引用计数：" << p1.use_count() << endl; // 输出 3

    // 3. 访问资源
    cout << *p2 << endl; // 10
    *p3 = 20;
    cout << *p1 << endl; // 20（所有 shared_ptr 指向同一资源）

    // 4. 重置：p3 放弃所有权，引用计数 -1
    p3.reset();
    cout << "p1 引用计数：" << p1.use_count() << endl; // 输出 2
```

#### （2）禁止的错误用法

```cpp
int* raw_ptr = new int(10);
shared_ptr<int> p1(raw_ptr);
shared_ptr<int> p2(raw_ptr); // 严重错误！两个 shared_ptr 独立管理同一原始指针
// 析构时 p1 和 p2 都会调用 delete raw_ptr，导致 double free 崩溃
```
