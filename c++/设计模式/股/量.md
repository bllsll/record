| 量比数值    | 结论                                              | 厦门空港实操举例                                             |
| ----------- | ------------------------------------------------- | ------------------------------------------------------------ |
| **＜0.8**   | 明显缩量（交易超清淡，买卖的人少）                | 厦门空港量比 0.6，说明今天成交比平时少 40%，若在 17.9 元支撑位，就是抛压快没了 |
| **0.8-1.2** | 平量（和平时成交差不多，无参考意义）              | 量比 1.0，成交和近 10 天平均一致，单独看没用                 |
| **1.2-2.0** | 温和放量（交易变活跃，资金开始进场 / 出场）       | 量比 1.8，成交比平时多 80%，若在压力位 18.9 元冲高，就是有资金尝试突破 |
| **＞2.0**   | 明显放量（交易很活跃，多空分歧大 / 资金大幅动作） | 量比 2.5，成交比平时多 1.5 倍，若在支撑位放量红盘，就是资金主动接盘 |
| **＞5.0**   | 极端放量（天量，少见，多空激烈博弈）              | 量比 5.8，成交是平时 5 倍以上，若在 18.9 元压力位，大概率是主力出货 / 抢筹 |

**核心作用：\**直接告诉你今天交易比平时活跃还是清淡\**，数值会实时更新（每笔交易后变一次），在行情软件的**分时图界面 ** 一眼就能找到（买一卖一旁边 / 成交量上方）。

 缩量：交易清淡，买卖的人少

 放量：交易活跃，买卖的人多（分 “有效放量” 和 “天量”，新手先看有效放量）

### 3. 关键：量比的**分时走势**比单个数更重要

量比不是固定的，开盘会很高（因为刚开盘成交集中），然后慢慢回落，正常走势是 **“高开低走，逐步平稳”**：

- 正常情况：9:30 开盘量比可能 10+，10 点后慢慢跌到 1 左右，代表成交回归正常；
- 异常情况：开盘量比低，**盘中突然快速拉升（比如从 0.7 涨到 2.0）**，代表突然有资金进场，是真实放量；反之，量比从 2.0 跌到 0.5，是突然缩量，交易变冷。
